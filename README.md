# Desafio Técnico — Estilização de Botões com CSS

Projeto criado para praticar CSS através de três níveis de dificuldade, 
todos focados na estilização e animação de botões.

## Níveis

- `iniciante/` — Estilização básica com hover e border-radius animado
- `intermediario/` — Gradiente, transições e estado focus
- `avancado/` — Gradiente avançado, border-image e transform com scale

## Tecnologias utilizadas

- HTML5
- CSS3

## Como visualizar

Clone o repositório e abra o arquivo `index.html` 
de cada pasta no navegador.
